# 开发者构建说明

普通用户不需要看这个文件。只有重新生成 exe 时才需要。

## 构建 exe

在 Windows PowerShell 中运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\build_windows_exe.ps1
```

会生成：

```text
dist\codex-api-switch.exe
dist\deepseek-responses-relay.exe
```

构建完成后，把这两个 exe 复制到项目根目录：

```powershell
copy .\dist\codex-api-switch.exe .\codex-api-switch.exe
copy .\dist\deepseek-responses-relay.exe .\deepseek-responses-relay.exe
```

## 可删除的临时文件

这些都是 PyInstaller 生成物，可以删除：

```text
build\
build-x64\
dist\
dist-x64\
__pycache__\
*.spec
```
